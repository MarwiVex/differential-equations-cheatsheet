# Ordinary Differential Equations Cheatsheet

This directory contains the English version of the cheatsheet, designed in LaTeX. It serves as an ideal reference material for students and researchers in mathematics, engineering, and physics. The document covers everything from basic first-order equations to Laplace transforms and systems of ODEs.

---

## Core Content

* **Identification Guide:** Resolution strategies based on standard forms.
* **First-Order ODEs:** Separable, Linear (Integrating Factor), Exact, Bernoulli, and Riccati equations.
* **Second-Order ODEs:** Constant and variable coefficients, Undetermined Coefficients, and Variation of Parameters.
* **Systems of ODEs:** Eigenvalue methods for homogeneous systems.
* **Laplace Transforms:** Fundamental theory, initial conditions, partial fractions, special functions, and a comprehensive transform table.
* **Visual Guide:** ODE Solution Strategy Decision Tree.

---

## Usage and Compilation

If you wish to build the PDF document (`ODE_Cheat_Sheet.pdf`) directly from the source code (`ode_cheatsheet.tex`), you can use your preferred LaTeX environment or run the following command in your terminal using `pdflatex`:

```bash
pdflatex ode_cheatsheet.tex
```

*Note: Depending on your setup, you may need to compile the document twice to ensure all cross-references and formatting are generated correctly.*
