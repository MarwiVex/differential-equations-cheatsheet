## English Overview
This repository contains a comprehensive cheatsheet for Ordinary Differential Equations (ODEs), designed in LaTeX. It serves as a quick reference guide for identifying, classifying, and solving various types of differential equations and systems.

**Main Topics:**
* First-Order ODEs and Techniques (Separable, Linear, Exact, Bernoulli, Riccati).
* Second-Order ODEs (Homogeneous, Non-Homogeneous, Cauchy-Euler).
* Systems of Linear First-Order ODEs.
* Laplace Transforms and Applied Analysis.

---

## Resumen en Español
Este repositorio contiene un formulario exhaustivo de Ecuaciones Diferenciales Ordinarias (EDO) diseñado en LaTeX. Sirve como una guía de referencia rápida para identificar, clasificar y resolver diversos tipos de ecuaciones y sistemas diferenciales.

**Contenido Principal:**
* Ecuaciones de primer orden y técnicas de resolución (Separables, Lineales, Exactas, Bernoulli, Riccati).
* Ecuaciones de segundo orden (Homogéneas, No Homogéneas, Cauchy-Euler).
* Sistemas de EDO lineales de primer orden.
* Transformadas de Laplace y análisis aplicado.
