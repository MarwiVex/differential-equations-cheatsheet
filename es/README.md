# Formulario de Ecuaciones Diferenciales Ordinarias

Este directorio contiene la versión en español del formulario, diseñado en LaTeX. Es un material de referencia ideal para estudiantes e investigadores en áreas de matemáticas, ingeniería y física. El documento abarca desde ecuaciones básicas de primer orden hasta transformadas de Laplace y sistemas de EDO.

---

## Contenido Principal

* **Guía de Identificación:** Estrategias de resolución basadas en formas estándar.
* **EDO de Primer Orden:** Ecuaciones separables, lineales (factor integrante), exactas, de Bernoulli y de Riccati.
* **EDO de Segundo Orden:** Coeficientes constantes y variables, coeficientes indeterminados y variación de parámetros.
* **Sistemas de EDO:** Métodos de valores y vectores propios para sistemas homogéneos.
* **Transformadas de Laplace:** Teoría fundamental, condiciones iniciales, fracciones parciales, funciones especiales y una tabla exhaustiva de transformadas.
* **Guía Visual:** Árbol de decisión para la estrategia de solución de EDO.

---

## Uso y Compilación

Si deseas compilar el documento PDF (`Formulario_EDO.pdf`) directamente desde el código fuente (`formulario_edo.tex`), puedes utilizar tu entorno LaTeX preferido o ejecutar el siguiente comando en la terminal usando `pdflatex`:

```bash
pdflatex formulario_edo.tex
```

*Nota: Dependiendo de tu configuración, es posible que necesites compilar el documento un par de veces para que las referencias internas y el formato se ajusten correctamente.*
